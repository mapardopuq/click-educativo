# Publicación de la web Click Educativo

Esta carpeta contiene la página web estática de Click Educativo lista para publicar en GitHub Pages.

## Archivos principales

- `index.html`: estructura de la página.
- `styles.css`: estilos visuales.
- `script.js`: menú móvil.
- `assets/`: imágenes optimizadas de la web.
- `.nojekyll`: evita procesamiento innecesario en GitHub Pages.

## Publicar en GitHub Pages

1. Entra a GitHub e inicia sesión.
2. Crea un repositorio nuevo llamado `click-educativo`.
3. Déjalo como `Public`.
4. Sube todos los archivos de esta carpeta, no subas el ZIP completo.
5. Verifica que `index.html` quede en la raíz del repositorio.
6. Entra a `Settings > Pages`.
7. En `Build and deployment`, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
8. Guarda.
9. La web debería publicarse en una URL similar a:
   `https://TU-USUARIO.github.io/click-educativo/`

## Recomendación

Antes de publicar, revisar que los botones de WhatsApp, correo e Instagram/Facebook abran correctamente.
